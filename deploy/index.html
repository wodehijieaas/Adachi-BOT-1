# 应用部署

TIP

本文档仅介绍 Linux 环境下的部署方式
不论从什么版本更新至 v2.2.0+，都请重新 clone 并重新配置

# Docker

# 安装

git clone https://github.com/SilveryStar/Adachi-BOT.git
cd Adachi-BOT

# 手动创建配置文件
# 根据右上角「配置」选项中的信息配置 setting.yml 和 cookies.yml
mkdir config && cd config
touch setting.yml commands.yml cookies.yml

# 初次启动需在宿主机完成设备验证
# 要求宿主机 Node.js version >= 12.16
npm install yaml oicq
npm run login

# 设备验证完成后，启动 docker
docker-compose up -d

# 更新

# 注意需在 Adachi-BOT 根目录中运行命令
docker-compose stop
git pull
docker-compose start

# Forever

# 安装

# 注意：此方法在 Windows 环境中不可用
git clone https://github.com.cnpmjs.org/SilveryStar/Adachi-BOT.git
cd Adachi-BOT
npm install -g forever

# 初次运行将生成配置文件模板
# 根据右上角「配置」选项中的信息配置 setting.yml
npm start

# 确保你已经在 56379 端口上运行 Redis 数据库
# 不会启动数据库的可以参考右上角「FAQ」中的信息
npm run serve

# 更新

forever list
# 停止上面输出的进程的 pid
forever list pid
git pull
npm run serve
